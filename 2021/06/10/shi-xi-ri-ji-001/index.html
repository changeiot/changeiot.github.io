ElementUI中Collapse组件的使用

ElementUI--Collapse
Web Development
发布日期:   2021-06-10
文章字数:   686
阅读时长:   2 分

安装ElementUI

在Vue项目中使用ElementUI的组件，首先第一步，安装ElementUI组件到项目中，这里使用npm命令安装：

npm i element-ui -S

引入ElementUI

然后在项目中引入，一般在项目中采取全局完整引入的方式。
这种方式的好处是，不需要再后续开发中不断的按需引入，省去了不少繁琐的代码，省心。

import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});

具体的引用和使用方式可以参考官方文档，官方文档永远是最友好的伙伴儿~

开始使用ElementUI组件

在需要使用Collapse组件的vue页面中使用ElemtnUI中提供的组件。以Collapse组件为例。

ElementUI组件使用之 Collapse

HTML部分

<el-collapse v-model="activeNames" @change="handleChange">
  <el-collapse-item title="第一个折叠行" name="1">
    <div>hello~</div>
    <div>I'm a component of ElmentUI.</div>
  </el-collapse-item>
  <el-collapse-item title="第二个折叠行" name="2">
    <div>Can you guess who I am? </div>
    <div>I can collapse or unfold some content while I was clicked</div>
  </el-collapse-item>
  <el-collapse-item title="第三个折叠行" name="3">
    <div>Yeah, how clever you are</div>
    <div>I am a Collapse component</div>
  </el-collapse-item>
</el-collapse>
  1. 其中的 title 是每一个折叠项的标题。
  2. 可以看到每一个折叠项 el-collapse-item标签内都有两个div标签，实际上这个div标签可以替换成合适的ElementUI的其他组件，如表格等等

JavaScript部分

<script>
  export default {
    data() {
      return{
        activeNames: ['1']
      };
    },
    methods: {
      handleChange(val) {
        console.log(val);
      }
    }
  }
</script>

  1. JavaScript 部分中，activeNames的值为默认值，即页面打开之后还未更新修改之前的时候 activeNames 的值。值为 “1” ，那么页面会默认展开第一个折叠行的内容。

  2. 其中， v-model=”activeNames”, activeNames 是绑定了 name 的值，如果 activeNames 为 “1”, 那么 name 为”1”的 那个 < el-collapse-item>就会被展开。

CSS部分

CSS可以按需调整。

需要注意的是，在ElementUI组件中，有不少组件官方已经提供了一些默认的样式，不需要写在CSS中，可以直接嵌套写在HTML部分中。
